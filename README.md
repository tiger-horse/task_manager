# 利用方法
1.ユーザーのアカウントを登録する

2.ルームを作成する

3.タスクを入力してカレンダーに反映する

4.タスクが終わったら未を押すと文字が薄くなる

5.カレンダーにあるタスクをクリックすると詳細に遷移してコメントを書くことができる
